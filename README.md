# 🏦 Loan Default Prediction System

An end-to-end Machine Learning project that predicts whether a customer is likely to default on a loan. This project includes data preprocessing, feature engineering, model training, and a deployed Streamlit web application for real-time predictions.

---

## 🚀 Features

- 🔍 Data Preprocessing & Cleaning  
- 🧠 Feature Engineering  
- ⚡ XGBoost Model for High Performance  
- 🌐 Streamlit Web App for Real-time Predictions  
- 📊 Probability-based Risk Analysis  

---

## 🧩 Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn  
- XGBoost  
- Streamlit
